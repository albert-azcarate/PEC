# 32-Bit CPU Implemented in VHDL

This repository contains the source code and design files for a 32-bit CPU implemented in VHDL. The project aims to create a fully-functional CPU that can execute basic instructions of SISA instruction set and support memory access, with a focus on simplicity and ease of understanding.

## Getting Started

To get started with the CPU design, clone the repository to your local machine and open the design files in a VHDL development environment. The top-level file is `cpu.vhd`, which contains the entity and architecture definitions for the CPU.

## Modifying the Design

The CPU design can be modified as needed to meet your specific requirements. For example, you can add additional instructions or modify the memory access functionality. However, please note that any modifications to the design should be thoroughly tested before implementation.
